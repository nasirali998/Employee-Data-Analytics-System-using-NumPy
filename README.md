# 📊 Employee Data Analytics System using NumPy

A professional Python project that analyzes employee data using **NumPy**. This project demonstrates the use of essential NumPy functions for performing salary, performance, and experience analysis.

## 🚀 Features

* 📌 Employee Salary Analysis
* 📌 Performance Analysis
* 📌 Experience Analysis
* 📌 Total & Average Salary
* 📌 Highest & Lowest Salary
* 📌 Best & Lowest Performer
* 📌 Salary Sorting (Ascending & Descending)
* 📌 Employee Filtering using Boolean Indexing
* 📌 Employee Classification using `np.where()`
* 📌 Unique Experience Analysis using `np.unique()`
* 📌 Professional Console Report

## 🛠️ Technologies Used

* Python 3
* NumPy

## 📚 NumPy Concepts Covered

* NumPy Arrays
* Shape, Size, Dimension & Data Type
* Array Slicing
* `np.sum()`
* `np.mean()`
* `np.max()`
* `np.min()`
* `np.argmax()`
* `np.argmin()`
* `np.sort()`
* `np.argsort()`
* Boolean Indexing
* Logical Operators (`&` and `|`)
* `np.where()`
* `np.unique()`
